# Musical-Key-Detection_System
Estimates the key of the song being played with live input from an or multiple instruments.

Hardware
- Elegoo board
- 4 digit 7 Segment display
- Computer microphone
- Resistors
- Wires
- Bread Board

Model
- Skey

Software
- Arduino IDE 2.3.6
- Python 3.11.5
